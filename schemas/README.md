# Swagger Schemas

Offline copies of Qubic RPC Swagger specs live under `schemas/swagger/`:

- `archive.swagger.json`
- `qubic-http.swagger.json`
- `query_services.swagger.json`
- `stats-api.swagger.json`

These were downloaded from https://qubic.github.io/integration/Partners/swagger/. Refresh them periodically via curl and replace the JSON files here so codegen/tests have stable inputs.
