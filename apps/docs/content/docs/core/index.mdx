`@qubickit/core` gives you the primitives that every other QubicKit package builds on: typed transports, schema-backed clients, transaction tooling, identity helpers, watchers, SSE clients, cache adapters, and diagnostics. Use this page as the jumping-off point for the rest of the Core docs.

## Install only what you need

```bash
# core transports + tooling (ready today)
bun add @qubickit/core

# higher-level packages (add later as they ship)
bun add @qubickit/sdk
bun add @qubickit/cli
bun add @qubickit/web
```

Head back to the [Transports & clients](/docs/core/clients) page once you install the package and want to wire it into your project.
