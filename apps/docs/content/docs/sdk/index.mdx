## Install alongside core

```bash
bun add @qubickit/core @qubickit/sdk
# optional: add @qubickit/cli or @qubickit/web when you need them
```

- The SDK reuses the same transports from `@qubickit/core`, so you always get typed clients and transaction tooling.
- Install only what you need; higher-level packages share the Bun toolchain and adapters.

## Roadmap snapshot
| Phase | Highlights |
| --- | --- |
| **Phase 0** | Shared Bun toolchain, adapters, event emitters, watcher channels, basic docs. |
| **Phase 1** | Wallet manager, session tokens, caching/throttling session client, tick/balance/transfer watchers. |
| **Phase 2** | Contract registry + generator, transfer orchestration, persistence adapters, command queue. |
| **Phase 3** | Multi-wallet limits, telemetry/logger hooks, sync daemon, typed SDK errors, doc deep dives. |

The current release bundles the Phase 0–3 features above; future phases will focus on compatibility matrices, telemetry exports, and versioned references.
